一个基于 Java（业务逻辑与路由）和 Python（AI模型推理）的Web图像处理系统。支持图像超分辨率（×2/×4）、色彩增强与智能降噪。所有用户数据（如处理历史）均保存在浏览器本地（IndexedDB），无需后端数据库。

核心特点：这是一个无后端数据库的设计。用户产生的所有数据都保存在其自己的浏览器本地，完全私有。后端（Java/Python）仅提供API和计算服务，不存储任何用户数据。
